# 🎓 Проекты Школы 21

## 📌 Описание
Этот репозиторий содержит мои решения проектов в рамках интенсива **Школа 21** (Сбер).

Все проекты выполнены на языке **C** в соответствии со стандартами структурного программирования и стилем кода Google.

---

## 📁 Список проектов

| Проект | Тема | Файлы |
|--------|------|-------|
| **D01** | Linux и Git | `src/sort.c`, `src/maxmin.c` и др. |
| **D05** | Игра Pong | `src/pong.c`, `src/pong_interactive.c` |
| **D07** | Указатели и массивы | `src/maxmin.c`, `src/stat.c`, `src/search.c` |
| **D08** | Динамическая память и матрицы | `src/matrix.c`, `src/det.c`, `src/invert.c` |
| **D09** | Многофайловые проекты, строки | `src/s21_string/*`, `src/text_processor.c` |
| **D10** | Структуры и колбэки | `src/list.c`, `src/stack.c`, `src/bst.c` |
